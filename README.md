Este projeto foi desenvolvido como atividade prática da disciplina de Banco de Dados para Tecnologia da Informação, do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de implementar um sistema delivery, permitindo gerenciar clientes, entregadores, pedidos, cardápio e entregas, 
utilizando um modelo relacional em MySQL.

Para o desenvolvimento foi Utilizado o MySQL Workbench, incluindo o modelo lógico e a implementação das tabelas com chaves primárias e estrangeiras.

Status: Projeto concluído e utilizado somente para fins acadêmicos.
Data: 19/04/2025.
